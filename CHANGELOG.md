# Changelog

Все заметные изменения проекта будут описаны в этом файле.

Формат версий: `x.y.z`
- `x` — крупное обновление, меняющее архитектуру или поведения; версии с одинаковым `x` остаются максимально стабильными.
- `y` — минорные важные рабочие обновления, добавляющие совместимые возможности.
- `z` — патчи, фиксы и результаты тестирования, не влияющие на совместимость API.

## [1.0.0] - 2025-10-29
- Базовая структура SandboxMCP (FastAPI + Caddy + policy + scripts + systemd).
- Опциональный стек визуального окружения (Xvfb/XFCE/x11vnc/noVNC) с make-целями и идемпотентным установщиком.
- Скрипты для запуска MCP (Playwright, Docker, Telegram) и bootstrap среды.
- Документация по переменным `.env`, в том числе подробный гайд по настройке Telegram MCP и без-GPU режиму.
- FastAPI эндпоинты `/health`, `/cua/health`, `/policy` и проверка CUA Computer Server.
- Исправлен health-check CUA (используется `/status`) и настроен Openbox wrapper для корректного отображения в noVNC.
- Автоматический запуск `xterm` в сессии Openbox, чтобы избежать «чёрного экрана» в noVNC.
